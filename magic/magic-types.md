# Magic types

This page breaks magic into distinct forms.

Use one section per type so readers can compare source, method, risk, and social meaning without guessing.

## For each magic type

Capture:

* Name
* Source of power
* Typical users
* What it can do
* Limits, costs, or fallout
* Public reputation

## Cross-link every type

Each magic type should connect back to:

* Species that use it
* Regions where it is strongest
* Events in history that changed it
* Other magic types it interacts with

{% hint style="info" %}
If two magic types can overlap, note where they blend and where they conflict.
{% endhint %}
