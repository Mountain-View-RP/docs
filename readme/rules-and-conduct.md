# Rules & Conduct

## Rules & Conduct

Use this page for the rules every player must follow.

Lead with the non-negotiables.

### Core rules

Add the rules that apply at all times.

Examples:

* Respect staff rulings
* No godmodding or powergaming
* Keep IC and OOC separate

### RP standards

Define what good play looks like here.

Cover pacing, scene consent, collaboration, and continuity.

### Safety and boundaries

State how players set limits and raise concerns.

Explain how staff handle issues.

### Enforcement

Set expectations for warnings, removals, and appeals.

Keep this section direct.

{% hint style="warning" %}
If a rule affects access, approval, or bans, state it plainly.
