# Regions overview

Regions define scale.

Start broad, then narrow into the city. Mountain View should feel connected to a larger world, not isolated from it.

Mountain View is an AU of current-day Sydney, so local geography, infrastructure, and civic texture can echo the modern city while still serving the setting's own canon.
